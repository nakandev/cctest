C Compiler Tests
================

Usage
-----

    ./runner.sh [options...]

options:
    --suite <DIR>           TestSuite root directory
    --work <DIR>            Working directory
    --cc <FILE>             Target Compiler
    --exec <FILE>           Target Executer
    --cflags <STR>          Target CFLAGS
    --ldflags <STR>         Target LDFLAGS
    --testlist <FILE>       Tesetlist file
